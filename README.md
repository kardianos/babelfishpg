# Babelfish - SQL Server behavior on PostgreSQL

Image for [Babelfish for PostgreSQL](https://babelfishpg.org/).

Currently builds BF v3.1.0 for PG 15.2.

## RUNTIME

 * Persistent data stored at `/data`.
 * PostgreSQL runs as user `postgres`.
